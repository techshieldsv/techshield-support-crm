# URLs para enviar a cliente y equipo

Reemplaza `https://tu-sitio.netlify.app` por la URL real de Netlify.

## Acceso general

- Inicio / Dashboard: `https://tu-sitio.netlify.app/#dashboard`
- Login demo: `https://tu-sitio.netlify.app/#login`

## URLs para cliente

- Portal del cliente: `https://tu-sitio.netlify.app/#portal`
- Crear / revisar tickets: `https://tu-sitio.netlify.app/#tickets`
- Base de conocimiento: `https://tu-sitio.netlify.app/#knowledge`
- Reportes: `https://tu-sitio.netlify.app/#reports`

## URLs para equipo interno

- Admin / Dashboard: `https://tu-sitio.netlify.app/#dashboard`
- Tickets soporte: `https://tu-sitio.netlify.app/#tickets`
- CRM comercial: `https://tu-sitio.netlify.app/#crm`
- Ajustes: `https://tu-sitio.netlify.app/#settings`
